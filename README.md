# GEOCA$HING - Flask Boilerplate App

 * SQLAlchemy
 * Flask (w/ flask-script)
 * WTForms
 * webassets
 * Bower
 * SemanticUI

# Configuration

```console
$ export GEOCASHING_CONFIG=/path/to/app.conf
```

# Initializing

```console
$ python manage.py bower
```

```console
$ python manage.py createdb
```

# Running

```console
$ python manage.py runserver
```
